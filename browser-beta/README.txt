Welcome!

The Corpus Browser allows users to load conversation data into a 
database, and create custom annotation columns. These annotations can then 
be used to run simple machine learning.

Folder structure must be:

CorpusBrowser-xxx.py
AUDIO
DATABASE
REQUIRED

Created in Python v3.6

Required Modules: (Note some are built in, some are not)

getpass, datetime, sqlite3, pygame, time, 
csv, platform, tkinter, numpy, nltk, sklearn
PIL, ttk
